Для каждого урока 

Перейти в папку с уроком и выполнить команды:

npx create-react-app myapp

cd myapp

npm start

-------------

CTRL + CTRL - остановка сервера

-----------------------

node_modules - она занимаеь 99% места. Это и есть сама библиотека. При выкладывании работ ее загружать не нужно.
Ее можно удалить, а чтобы зарузить заново, для этого нобходимо:
перейти в текущую директорию и в терминале ввести 'npm install'

public - хранит стартовую разметку, будем использовать редко. Хранятся файлы, которые не будут меняться

src - с ней мы и работаем.
